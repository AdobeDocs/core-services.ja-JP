# Experience Cloudサービスと管理

Experience Cloud には、アドビが提供するデジタルマーケティングソリューションおよびサービスが統合されています。また、Experience Cloud は、クラウドソリューションやコアサービスにすばやくアクセスできる直感的なインターフェイスでもあります。

![Experience Cloud](assets/cloud-pulldown.png)

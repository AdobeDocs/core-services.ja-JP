# Activation

Activationには、起動、Adobe、Dynamic Tag Management（DTM）、Triggersという最新のタグ管理製品が含まれています。

## Experience Platform Launch {#section_05DF74E9EAC94518A155095C65F54568}

パーソナライズされたカスタマーエクスペリエンスの強化に必要な分析タグ、マーケティングタグ、広告タグのデプロイと管理をおこないます。

[Launch, by Adobe](https://marketing.adobe.com/resources/help/en_US/experience-cloud/launch/) に関するページを参照してください。

## Dynamic Tag Management {#section_C7E000EEF3E6459FB4B6D4A8960DD8F0}

## Experience Cloud デバッガー {#section_682FCF8A22ED47D0ACF94204AB94B444}

Chrome用Adobe Experience Cloud Debugger拡張機能によりWebページが調べられ、Experience Cloudソリューションの実装方法に関する問題を見つけることができます。

ヘルプについては [、Experience Cloudデバッガー](https://marketing.adobe.com/resources/help/en_US/experience-cloud-debugger/) を参照してください。

# スキーマの更新（属性の削除）

スキーマの属性を削除して置換する方法。

1. [!UICONTROL Edit Customer Attribute Source] ページで、「購読を設定」をクリックします。

1. **[!UICONTROL Target]**&#x200B;または&#x200B;**[!UICONTROL Analytics]**&#x200B;のマッピングされた値を削除します。

1. [更新されたフィールドを含む新しいデータファイルをアップロードします](t-crs-usecase.md)。
